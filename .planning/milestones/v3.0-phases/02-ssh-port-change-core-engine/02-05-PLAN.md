Create Section + SectionHeader as named exports. Section replaces ad-hoc div grouping patterns used across screens.

Section props:
- `title?: string` — section title (renders SectionHeader if provided)

- `action?: ReactNode` — optional action slot (e.g., button in header)

- `onToggle?: () => void`
- `className?: string`

- Section wrapper: flex flex-col gap-[var(--space-4)]

- Title: text-[var(--font-size-lg)] font-[var(--font-weight-semibold)] text-[var(--color-text-primary)] tracking-[var(--tracking-tight)]
- Description: text-[var(--font-size-sm)] text-[var(--color-text-muted)] mt-1
- Collapsible chevron: Lucide ChevronDown, rotate-180 when open, transition duration-[var(--transition-fast)]
