**3. index.css — remove 2 !important overrides (per D-05):**

Line 40: `.window-control-close:hover { color: #fff !important; }`
- Remove `!important`. The rule already has sufficient specificity (`.window-control-close:hover` is more specific than `.window-control-btn:hover`). The `.window-control-btn:hover` rule on line 37 sets `color: var(--color-text-primary)` — but `.window-control-close:hover` is a more specific selector (compound class), so it wins without `!important`.
- Also replace hardcoded `#fff` with `#fff` (keeping it as-is — white on red close button is a platform convention, not a theme token).

Line 77: `input::-webkit-credentials-auto-fill-button { display: none !important; }`
- This is a browser pseudo-element override that fights with browser UA stylesheet. Increase specificity by nesting under a parent selector. Replace with:
```css
[data-theme] input::-webkit-credentials-auto-fill-button {
  display: none;
}
```
The `[data-theme]` attribute is always present on the root element, so this adds specificity without changing behavior. This selector beats the UA stylesheet because attribute selectors have higher specificity than pseudo-element defaults.

```markdown
## Tint Tokens

Полупрозрачные фоновые тинты для статусных и акцентных элементов. Определены для обеих тем — dark использует 400-tier цвета (ярче), light использует 600-tier (контрастнее).

### Naming Convention
`--color-{category}-tint-{opacity}` where opacity is 03-50 (percentage * 100).

### Categories
| Category | Dark Base | Light Base | Opacities |
|----------|-----------|------------|-----------|
| accent | accent-500 (2d7a76) | accent-600 (236260) | 06, 08, 10, 15, 20, 30, 50 |
| success | success-500 (10b981) | success-600 (059669) | 06, 08, 12, 15, 25, 30, 40 |
| warning | warning-500 (f59e0b) | warning-600 (d97706) | 05, 08, 12, 15, 20, 30, 40 |
| danger | danger-500 (ef4444) | danger-600 (dc2626) | 03, 08, 10, 15, 25 |

### Overlay Tokens
| Token | Value | Usage |
|-------|-------|-------|
| `--color-overlay-40` | rgba(0,0,0,0.40) | Modal backdrop (light) |
| `--color-overlay-50` | rgba(0,0,0,0.50) | Modal backdrop (dark) |

### Usage
```css
background-color: var(--color-success-tint-08);  /* status bg */
border: 1px solid var(--color-warning-tint-20);   /* status border */
box-shadow: 0 8px 24px var(--color-accent-tint-30); /* glow */
```
```
