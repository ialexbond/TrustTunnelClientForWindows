## Components Delivered

### dirtyTracker.ts
Pure utility — `isDirty(initial, current)` shallow+array diff. `createSnapshot()` filters undefined. Used for D-9 dirty-state warning in UserModal Edit mode.

### PasswordRotationPrompt
D-7 password rotation sub-component. Renders inline when `isOpen=true`. Includes:
- Warning banner (always shown): «Старые deeplink'и перестанут работать»
- New password input with Shuffle regen + clearable
- Confirm (danger variant) + Cancel buttons
- Error banner for backend rotation errors
- `validatePassword()` matching UsersAddForm rules

### CertificateFingerprintCard
D-6 cert pinning sub-component. 4 states (idle / loading / error / success).
- `server_fetch_endpoint_cert` invoke on fetch click
- SHA-256 fingerprint display in mono font
- `onFingerprintLoaded(derB64, fingerprint)` for parent TLV embedding
- Storybook `_force*` props bypass backend

### DnsUpstreamsInput
D-4 dns_upstreams (TLV 0x0D) textarea input. Per-line validation:
- Rejects shell metacharacters (mirrors server-side validate_dns_list)
- Allows IPv4, IPv4+port, hostname, hostname+port
- `aria-invalid` + per-entry error display
- Exported `isValidDnsEntry` for unit testing

### UserModal
D-1 through D-9 unified Add/Edit modal. Two always-visible sections:
- **Учётные данные**: username + password (Add) / readonly pw + rotate button (Edit)
- **Параметры deeplink**: anti-DPI toggle, display name, custom SNI, upstream protocol, skip-verify toggle, cert pinning, DNS upstreams, CIDR picker

Add mode: `server_add_user_advanced` with all params.
Edit mode: `server_get_user_config` load → `server_update_user_config` save.
Dirty banner: `isDirty(initialSnapshot, currentDeeplinkSnapshot)` → warning.

### UsersSection (refactor)
- Plus-icon in CardHeader → UserModal Add mode (D-2)
- Settings/Gear icon per row → UserModal Edit mode (D-3)
- FileText + Trash unchanged
- All `useCallback` hooks moved above `if (!serverInfo) return null` guard (rules-of-hooks compliance)
- Updated 5 outdated Phase-14 tests to Phase-14.1 assertions
