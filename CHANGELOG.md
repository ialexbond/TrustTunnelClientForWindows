# Changelog

Все значимые изменения проекта документируются в этом файле.

## [1.2.1] — 2025-03-18

### Добавлено

- **Автообновление** — встроенный механизм обновления через GitHub Releases
  (скачивание ZIP, распаковка, автоматический перезапуск)
- **Индикатор VPN-статуса** в шапке приложения (цветная точка рядом с вкладками)
- **Требования к серверу** на экране приветствия (CPU, RAM)
- **Кастомные кнопки ±** для настройки MTU вместо нативных стрелок браузера
- **Сегментированный переключатель протокола** (HTTP/2 / HTTP/3) вместо выпадающего списка
- **Жёлтая кнопка** во время подключения/отключения — визуально отражает процесс,
  блокирует повторные нажатия
- **Плавная анимация** раскрытия/скрытия «Опасной зоны» (CSS max-height transition)
- **Автоскролл** к «Опасной зоне» при её раскрытии

### Изменено

- Убрана панель логов — упрощён интерфейс
- Убран верхний спиннер на экране установки — достаточно спиннеров в списке шагов
- Убраны пока что бессмысленные поля «Уровень логирования» и «Custom SNI» из настроек
- Кнопка «Сохранить» перемещена выше переключателей (Kill Switch и т.д.)
- «Опасная зона» теперь отображается как кнопка по центру внизу настроек
- Пароль SSH в «Опасной зоне» больше не подгружается из localStorage
  (ранее отображался в некорректном виде из-за base64)
- Режим VPN «Только избранное» переименован в «Напрямую»
- После установки VPN больше не происходит автоподключение —
  пользователь сам подключается через вкладку «Настройки»
- Обновлён README: актуальные функции, требования к серверу, roadmap

### Исправлено

- Сохранение конфигурации больше не затирает список исключений маршрутизации
- `.gitignore` корректно игнорирует все портативные сборки (wildcard-паттерн)

## [1.2.0] — 2025-03-15

### Добавлено

- Мастер установки VPN-сервера через SSH (пошаговый wizard)
- Панель маршрутизации с импортом/экспортом списков доменов
- Kill Switch, Anti-DPI, Post-Quantum криптография
- Системный трей с цветной иконкой (красная/зелёная)
- Сворачивание в трей при закрытии окна
- Режимы VPN: «Всё через VPN» и «Напрямую»

## [1.0.0] — 2025-03-01

### Добавлено

- Первый релиз: GUI-обёртка над trusttunnel_client (C++ sidecar)
- Подключение/отключение VPN
- Панель статуса с таймером uptime
- Выбор конфигурационного файла
