# Colors

Near-monochrome system. Restrained elegance. Two themes that feel like one cohesive family.

## Dark Theme Surfaces

| Token | Value | Usage |
|-------|-------|-------|
| `--color-bg-primary` | `#0d0d0d` | Window background |
| `--color-bg-secondary` | `#161616` | Sidebar, secondary panels |
| `--color-bg-surface` | `#1c1c1c` | Cards (always + border) |
| `--color-bg-elevated` | `#222222` | Modals, dropdowns |
| `--color-bg-hover` | `#2a2a2a` | Hover state |
| `--color-bg-active` | `#323232` | Active/pressed state |

## Text Colors

| Token | Dark | Light | Contrast (dark) | Contrast (light) |
|-------|------|-------|-----------------|------------------|
| `--color-text-primary` | `#f2f2f2` | `#161616` | 18.3:1 | 18.3:1 |
| `--color-text-secondary` | `#9a9a9a` | `#5a5a5a` | 7.4:1 | 6.6:1 |
| `--color-text-muted` | `#6e6e6e` | `#767676` | 4.6:1 | 4.5:1 |

All pairs meet WCAG 2.1 AA (4.5:1 minimum for normal text).

## Accent Reserved For

1. Primary action buttons (connect/apply/save)
2. Toggle control "on" state
3. Input :focus-visible ring
4. Active sidebar navigation indicator
5. Inline links within body text
6. Storybook accent swatches (this page)

Accent is **not** used for: headings, card borders, icons at rest, badges, status indicators, secondary buttons, background fills.
