## Removed in v3.0

The following glow effects are deprecated and set to `none`:
- `successGlow` -- was `0 0 12px rgba(16, 185, 129, 0.6)`
- `dangerGlow` -- was `0 0 12px rgba(239, 68, 68, 0.4)`
- `accentLogoGlow` -- was `rgba(99, 102, 241, 0.15)`

Replaced by opacity pulse animation for status dots (`--pulse-duration: 2s`).

## Focus Ring

| Token | Value | Usage |
|-------|-------|-------|
| `--focus-ring` | `0 0 0 2px var(--color-bg-primary), 0 0 0 4px var(--color-accent-interactive)` | All interactive elements via :focus-visible |

Double-ring pattern: inner ring matches background, outer ring is accent. Works on all surfaces.
