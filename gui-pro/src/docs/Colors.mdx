### Semantic mapping

| Token | Dark theme | Light theme |
|---|---|---|
| `--color-accent-interactive` | `accent-400` (#4d9490) | `accent-600` (#236260) |

**Hover = more lightness, Active = more darkness.** Symmetric per theme ‚Äî dark lightens –Ω–∞ hover, light darkens.

### When to use accent

‚úÖ **Do:**
- Primary buttons background
- Active tab indicator
- Selected state (radio, checkbox checked)
- Links inline
- Focus ring outer layer
- Brand elements (wordmark in TitleBar)
- Key interactive icons

‚ùå **Don't:**
- Body text (use `--color-text-primary`)
- Card backgrounds (use `--color-bg-surface`)
- Borders as decoration (use `--color-border`)
- Status feedback (use semantic status colors)

## Rules

### Core
- **10% accent rule** ‚Äî accent —Ç–æ–ª—å–∫–æ –¥–ª—è interactive elements. Max 10% visible surface.
- **Semantic tokens only** ‚Äî use `--color-status-connected`, not `--color-success-400` directly.
- **Theme parity** ‚Äî –∫–∞–∂–¥—ã–π semantic token –æ–ø—Ä–µ–¥–µ–ª—ë–Ω –∏ –≤ dark, –∏ –≤ light. –ù–∏–∫–æ–≥–¥–∞ –Ω–µ assumed.
- **No hardcoded hex** ‚Äî –≤—Å–µ colors —á–µ—Ä–µ–∑ tokens. –ò—Å–∫–ª—é—á–µ–Ω–∏—è documented (ErrorBoundary fallback, OS chrome).

### Status tier selection (rule)
- **Dark UI** (badges, icons, dots): `-400`
- **Light UI ‚â•18pt** (large badges, icons): `-600`
- **Body text / readable content (both themes):** `-500` midpoint ‚≠ê safe default

### Tailwind integration
- **Use Tailwind bracket notation** ‚Äî `text-[var(--color-text-primary)]`, `bg-[var(--color-bg-surface)]`
- **NEVER** Tailwind native color classes (`text-red-500`, `bg-blue-400`) ‚Äî –æ–Ω–∏ –Ω–µ —Å–≤—è–∑–∞–Ω—ã —Å tokens.
- **inline `style={{ color: "var(--color-*)" }}`** ‚Äî OK –¥–ª—è dynamic/conditional colors.

### Focus
- **Keyboard focus:** `focus-visible:shadow-[var(--focus-ring)]`
- **Active selection (not focus):** `0 0 0 2px var(--color-accent-500)` solid (StepBar pattern)

## Don'ts

- ‚ùå **Hardcoded hex** ‚Äî `color: "#ef4444"` –≤ style. –ò—Å–ø–æ–ª—å–∑—É–π `var(--color-danger-500)`.
- ‚ùå **Tailwind native** ‚Äî `text-red-500`, `bg-green-400`. –ò—Å–ø–æ–ª—å–∑—É–π bracket notation.
- ‚ùå **Inline rgba()** ‚Äî `rgba(16, 185, 129, 0.08)` custom. –ò—Å–ø–æ–ª—å–∑—É–π `var(--color-status-connected-bg)` –∏–ª–∏ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–π tint token.
- ‚ùå **Accent –¥–ª—è non-interactive** ‚Äî accent-colored text paragraph, card background, decorative border. 10% rule violation.
