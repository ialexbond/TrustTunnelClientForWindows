# Typography

Типографическая система TrustTunnel (Pro & Light).

**Три шрифта, три роли.** Все поддерживают кириллицу.

| Шрифт | Tailwind | Применение |
|---|---|---|
| **Geist Sans** | `font-sans` (default) | Весь UI: тексты, кнопки, заголовки, формы. |
| **Geist Mono** | `font-mono` | Только technical data: IP-адреса, порты, SHA-fingerprint, hex, deeplink TLV, лог-строки, timestamp в логах. |
| **Outfit** | `font-display` | Исключительно wordmark «TrustTunnel» (AboutPanel, TitleBar). Нигде больше. |

**Правило labels vs values:**
> Подпись (label) — в sans. Техническое значение (value) — в mono.
> «Ваш IP: 192.168.1.1» — «Ваш IP» sans, `192.168.1.1` mono.

---

## Шкала — UI (7 шагов)

Шаг 2px, всё чётное, минимум 12px.

<Typeset
  fontSizes={[12, 14, 16, 18, 20, 22, 24]}
  fontWeight={400}
  sampleText="Быстрая коричневая лиса / The quick brown fox"
  fontFamily='"Geist Sans", "Segoe UI", system-ui, -apple-system, sans-serif'
/>

## Шкала — Display (Geist Sans bold)

Два display-размера для wizard hero и splash / critical states. Шрифт — Geist Sans bold.

<Typeset
  fontSizes={[32, 40]}
  fontWeight={700}
  sampleText="Добро пожаловать / Welcome"
  fontFamily='"Geist Sans", "Segoe UI", system-ui, -apple-system, sans-serif'
/>

## Wordmark — Outfit 48px

Отдельный display-face. Применяется **только** для wordmark «TrustTunnel» (AboutPanel, TitleBar). В обычном UI Outfit не используется.

<Typeset
  fontSizes={[48]}
  fontWeight={700}
  sampleText="TrustTunnel"
  fontFamily='"Outfit", "Geist Sans", system-ui, -apple-system, sans-serif'
/>

## Веса

Четыре веса. Regular и Medium — для обычного текста, Semibold — для заголовков, Bold — для display.

**Regular (400)** — body, paragraph, description:

  fontSizes={[16]}
  fontWeight={400}
  sampleText="Быстрая коричневая лиса / The quick brown fox"

/>

**Medium (500)** — buttons, subtitle, active tab, chip labels:

| Token | Ratio | Tailwind | Where |
|---|---|---|---|

| Token | Value | Tailwind | Where |
|---|---|---|---|

| `--tracking-normal` | 0 | `tracking-normal` | Default body |
| `--tracking-wide` | 0.02em | `tracking-wide` | Buttons, UPPERCASE badges, keyboard shortcuts |

## Semantic classes — ЭТО главный инструмент

14 композитных классов. Каждый устанавливает **family + size + weight + line-height** (+ опционально tracking) одновременно.

**Используй их напрямую.** Не комбинируй `text-lg font-semibold leading-snug` вручную — для этого есть `text-title`.

| Class | Recipe | Use for |
|---|---|---|

| `.text-body` | 16 / regular / 1.5 / sans | **Default body** — параграф, input text, description |
| `.text-body-lg` | 18 / regular / 1.5 / sans | Emphasis body |
| `.text-subtitle` | 18 / medium / 1.35 / sans | Subtitle, emphasis, accordion trigger |
| `.text-button` | 14 / medium / 1 / sans / tracking-wide | Button labels |
| `.text-title-sm` | 20 / semibold / 1.35 / sans | Card titles |
| `.text-title` | 22 / semibold / 1.35 / sans | Section titles, modal headers |
| `.text-title-lg` | 24 / semibold / 1.35 / sans | Page / tab headings |
| `.text-display-sm` | 32 / bold / 1.2 / sans / tracking-tight | Wizard hero, welcome |
| `.text-display` | 40 / bold / 1.2 / sans / tracking-tight | Splash, critical error state |
| `.text-wordmark` | 48 / bold / 1.1 / **Outfit** / tracking-tight | Wordmark «TrustTunnel» only |
| `.text-mono` | 14 / regular / 1.5 / **mono** | IP, SHA256, hex, log line, TLV payload |
| `.text-mono-sm` | 12 / regular / 1.5 / **mono** | Log timestamps, caption-level tech data |

## Decision tree — если компонент не в таблице выше

1. **Meta, version, timestamp, count?** → `text-caption`
2. **Helper text, secondary, short hint?** → `text-body-sm`
3. **Основной параграф, input text, label?** → `text-body` (default)
4. **Emphasis body, subtitle?** → `text-body-lg` / `text-subtitle`
5. **Кнопка?** → `text-button`
6. **Card title?** → `text-title-sm`
7. **Section / modal title?** → `text-title`
8. **Page / tab heading?** → `text-title-lg`
9. **Wizard hero?** → `text-display-sm`
10. **Splash / critical state?** → `text-display`
11. **Wordmark «TrustTunnel»?** → `text-wordmark`
12. **Technical data (IP / SHA / hex / log)?** → `text-mono` (или `text-mono-sm` для compact)

## Rules

### Minimums

- **Body text:** минимум 12px. Всё что ниже — нечитаемо для кириллицы на Win11 HiDPI.
- **Display:** минимум 32px. Всё что 24 и ниже — это `text-title-lg`, не display.

### Maximums

- **Regular body:** максимум 18px (`text-body-lg`). Выше — это уже title или display.
- **Inline text:** максимум `text-title-lg` (24). Display — для изолированных hero-блоков, не inline.

### Font usage

- **Sans** — UI text. Default, ничего не надо писать.
- **Mono** — **только** technical data (IP, hex, SHA, log lines, numeric tabular data).
- **Display (Outfit)** — **только** wordmark «TrustTunnel». Нигде больше.

### Weight usage

- **Regular (400):** body, paragraph, description — 90% контента.
- **Medium (500):** buttons, subtitle, active state, labels в формах.
- **Semibold (600):** headings (title-sm, title, title-lg).
- **Bold (700):** display-only. Обычный текст в bold не пишем.

### Letter spacing

- **tracking-tight:** автоматически применяется в `.text-display-*` и `.text-wordmark`. Руками не надо.
- **tracking-wide:** автоматически в `.text-button`. Плюс — UPPERCASE badges / keyboard shortcuts (например `<kbd>Ctrl+C</kbd>`).
- **tracking-normal:** default для всего остального.

## Don'ts

- ❌ **`font-size < 12px`** — нечитаемо для кириллицы.
- ❌ **`font-mono` для body-текста** — mono предназначен для technical data.
- ❌ **Outfit** для обычного UI — только wordmark.
- ❌ **`font-[var(--font-weight-semibold)]`** — старый arbitrary-value pattern. Используй `font-semibold`.
- ❌ **`text-[var(--font-size-*)]`** — Tailwind генерирует `color:` вместо `font-size:`. Используй обычные классы или semantic (`text-body` etc.).
- ❌ **`style={{ fontSize: "var(--font-size-*)" }}`** — inline style. Используй Tailwind class.
- ❌ **Ручная комбинация `text-lg font-semibold leading-snug`** — есть semantic class (`text-subtitle`, `text-title`).
- ❌ **Bold (700) для обычного текста** — bold only для display/hero.

## Cyrillic

- Русский визуально шире английского на ~15%. Длинные слова («Переподключиться», «Синхронизация», «Деинсталлировать») проверяй на overflow.
- **Минимум размера для RU-текста:** 12px. Ниже кириллица теряет читаемость.
- Geist Sans, Geist Mono, Outfit — все три с полной поддержкой кириллицы.
- При локализации: если англ текст помещается на 14px, для русского — тот же размер или на шаг выше (14 → 16).
